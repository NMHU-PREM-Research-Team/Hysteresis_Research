# Data Availability

The full dataset used in this study is publicly available via Zenodo.

DOI: https://doi.org/10.5281/zenodo.17969954

The dataset is provided as a compressed archive. After extraction, the data
directory should have the following structure:

data/
├── waspwaisted/
├── gooseneck/
└── regular/

Each file corresponds to a single magnetic hysteresis loop and is provided as a
plain-text, tab-delimited (.dat) file.

This repository contains code only; raw data are hosted externally to ensure
long-term preservation and citability.
